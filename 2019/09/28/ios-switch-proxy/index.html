iOS快速设置代理

发表于 2019-09-28

问题

手动设置wifi的步骤非常长：

  1. 点击“设置”
  2. 点击“无线局域网”
  3. 点击已连接的wifi
  4. 点击“配置代理”
  5. 点击“手动”
  6. 定位服务器输入框，输入ip
  7. 定位端口输入框，输入port
  8. 点击“存储”

解决

用iOS上的Shadowrocket和Mac上的Charles来快速设置代理
iOS和Mac在同一wifi

  1. 启动Charles，假设代理地址为：192.168.0.100:8888
  2. Shadowrocket设置全局路由为代理
  3. Shadowrocket添加HTTP类型的节点
  4. Shadowrocket打开连接
